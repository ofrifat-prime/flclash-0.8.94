# CLAUDE.md — Правила для ИИ-ассистента

## Обязательный workflow
1. **Analyze** — изучи архитектуру, перечисли файлы, объясни риски
2. **Plan** — предложи план изменений, разбей на подзадачи
3. **Test** — опиши тесты (unit, widget, integration)
4. **Code** — только после одобрения плана и тестов

## Запрещено
- Писать код до завершения анализа
- Пропускать шаги
- Ломать существующий импорт профилей
- Менять бизнес-логику без явного запроса

## Обязательно
- Минимальные изменения
- Обратимость (rollback strategy)
- Комментарии в нетривиальных местах

## Контекст проекта
- FlClashR — форк FlClash (GPLv3)
- Целевая аудитория: обычные люди в России
- Ядро: Mihomo (Clash Meta)
- Фреймворк: Flutter + Riverpod + Freezed
- Язык по умолчанию: русский
- Главный принцип: простота > функциональность
